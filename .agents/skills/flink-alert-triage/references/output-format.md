# Output Format

Keep the response operational and short.

## Suggested Shape

- `【一眼】` one sentence with the action
- 2 to 4 bullets with the key evidence
- 1 to 3 numbered next steps
- optional links at the end

## Good Output Traits

- starts with the answer, not the background
- includes actual metrics or statuses
- makes clear whether the issue is stable, worsening, or recovered
- avoids repeating the same diagnosis in every follow-up alert
