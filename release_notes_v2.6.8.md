# SPX Helper v2.6.8

## 🎉 主要更新

- ✨ **优化日程卡片布局**：改进日历日程的展示效果
- 🔧 **优化展开/收起功能**：更流畅的日程详情查看体验
- 🚀 **简化更新检查**：直接显示 GitHub Releases 链接，方便快速更新
- 🛠️ **改进错误处理**：优化 GitHub API 调用和错误提示

## 📦 安装方法

1. 下载 `SPX_Helper_v2.6.8.zip` 文件
2. 解压到本地目录
3. 打开 Chrome，访问 `chrome://extensions/`
4. 开启"开发者模式"
5. 点击"加载已解压的扩展程序"
6. 选择解压后的文件夹

## 🔄 更新说明

如果你是从旧版本更新：
1. 建议先导出数据（设置 → 导出数据）
2. 移除旧版本扩展
3. 安装新版本
4. 导入之前导出的数据
