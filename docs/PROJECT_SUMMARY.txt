📦 项目名称: SPX Helper (值班助手)
📅 创建日期: 2025-11-14
🎯 项目目标: 为日常值班工作提供高效辅助工具
⚡ 技术栈: Chrome Extension Manifest V3 + 原生JS

✅ 1. 快速笔记
   - 即时记录值班事项
   - 自动时间戳标记
   - 本地持久化存储
   - 支持删除和查看历史

✅ 2. 智能提醒
   - 自定义提醒内容和时间
   - 系统级桌面通知
   - 实时倒计时显示
   - 支持取消未触发的提醒

✅ 3. 链接管理
   - 保存常用工作链接
   - 一键快速访问
   - URL有效性验证
   - 简单的增删操作

✅ 4. 值班统计
   - 实时值班计时
   - 笔记数量统计
   - 值班历史记录
   - 开始/结束管理

1️⃣ 安装扩展
   • 打开 chrome://extensions/
   • 开启"开发者模式"
   • 加载已解压的扩展程序
   • 选择 SPX_Helper 文件夹

2️⃣ 固定图标
   • 点击拼图图标 🧩
   • 找到"值班助手"
   • 点击图钉固定 📌

3️⃣ 开始使用
   • 点击工具栏图标
   • 开始值班
   • 记录笔记
   • 设置提醒
