# SPX Helper 快速安装指南

## 📦 安装步骤

### 方法一：拖拽安装（最简单）⭐

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 打开右上角的"**开发者模式**"开关
3. **直接拖拽** `SPX_Helper_v2.2.0.zip` 文件到扩展管理页面
4. 完成！扩展图标会出现在浏览器工具栏

### 方法二：解压安装

1. 下载 `SPX_Helper_v2.2.0.zip`，解压到一个**不会删除**的目录（如 `~/Applications/Chrome Extensions/`）
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 打开右上角的"**开发者模式**"开关
4. 点击"**加载已解压的扩展程序**"按钮
5. 选择刚才解压的 `SPX_Helper` 文件夹
6. 完成！扩展图标会出现在浏览器工具栏

**推荐使用方法一，更简单快捷！**

### 固定扩展图标（可选）

点击浏览器工具栏的拼图图标 🧩，找到 **SPX Helper**，点击📌图钉图标将其固定到工具栏。

## 🚀 快速开始

### 立即可用的功能
- ✅ **快速链接**：访问常用的 DataSuite 平台和工具

### 需要配置的功能

#### 📅 今日日程（Google Calendar）
首次使用需要配置 OAuth 认证，请按照以下步骤操作：

1. 在扩展中点击"**今日日程**"标签
2. 点击"**⚙️ 配置**"按钮，按照页面提示完成配置
3. 详细配置步骤请查看 `OAUTH_SETUP_QUICK.md` 文件

**这是一次性配置，完成后即可长期使用！**

## ⚠️ 重要提示

1. **不要删除**解压后的扩展文件夹，否则扩展将无法正常工作
2. 建议将扩展文件夹放在固定位置，如：
   - macOS: `~/Applications/Chrome Extensions/SPX_Helper/`
   - Windows: `C:\Program Files\Chrome Extensions\SPX_Helper\`
3. 所有数据（代办事项、自定义链接等）都保存在浏览器本地存储中
4. 更新扩展时，直接替换文件夹内容并在 `chrome://extensions/` 中点击刷新即可
