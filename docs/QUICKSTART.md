# ⚡ 快速启动指南

## 3分钟上手使用

### 步骤1: 加载扩展 (1分钟)

1. 打开Chrome浏览器
2. 地址栏输入: `chrome://extensions/`
3. 打开右上角 **"开发者模式"**
4. 点击 **"加载已解压的扩展程序"**
5. 选择 `SPX_Helper` 文件夹
6. 完成！

### 步骤2: 固定到工具栏 (30秒)

1. 点击浏览器工具栏的拼图图标 🧩
2. 找到 **"值班助手"**
3. 点击图钉 📌 固定

### 步骤3: 开始使用 (30秒)

1. 点击工具栏的值班助手图标
2. 点击 **"开始值班"**
3. 记录你的第一条笔记
4. 设置一个提醒
5. 完成！

## 注意事项

### ✅ 推荐做法
- 及时记录重要信息
- 定期设置巡检提醒
- 保存常用工作链接
- 开始值班时启动计时

### ⚠️ 注意事项
- 不要清除浏览器数据（会丢失笔记）
- 确保通知权限已开启
- 重要笔记建议额外备份
- 卸载前导出重要数据

## 疑难解答

### Q: 通知不弹出？
**A:** 检查以下设置：
1. Chrome设置 → 隐私和安全 → 网站设置 → 通知
2. 确保允许Chrome发送通知
3. 关闭勿扰模式

### Q: 数据丢失了？
**A:** 可能原因：
1. 清除了浏览器数据
2. 卸载了扩展
3. Chrome数据损坏

**预防措施:**
- 定期导出重要笔记（未来版本会添加此功能）
- 不要轻易清除浏览器数据

### Q: 扩展无法加载？
**A:** 检查步骤：
1. 确认开启了"开发者模式"
2. 检查文件夹是否完整
3. 查看是否有错误提示
4. 尝试刷新扩展

## 高级技巧

### 技巧1: 批量操作
在开发者工具控制台执行：
```javascript
// 导出所有笔记
chrome.storage.local.get(['notes'], result => {
  console.log(JSON.stringify(result.notes, null, 2));
});

// 清除所有数据（小心使用！）
chrome.storage.local.clear();
```

### 技巧2: 自定义提醒时间
修改默认时间：
1. 右键扩展 → 审查弹出内容
2. 在控制台修改默认值

### 技巧3: 批量添加链接
如果有很多链接要添加，可以在控制台批量执行：
```javascript
const links = [
  { name: '监控系统', url: 'https://monitor.com' },
  { name: '工单系统', url: 'https://ticket.com' }
];

chrome.storage.local.get(['links'], result => {
  const existing = result.links || [];
  const newLinks = links.map(link => ({
    id: Date.now() + Math.random(),
    ...link
  }));
  chrome.storage.local.set({ 
    links: [...existing, ...newLinks] 
  });
});
```
