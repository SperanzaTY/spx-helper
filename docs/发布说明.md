## 🚀 给同事的使用流程

### 1️⃣ 安装（5 分钟）
1. 解压 `SPX_Helper_v2.2.0.zip`
2. 在 Chrome 中访问 `chrome://extensions/`
3. 打开"开发者模式"
4. 点击"加载已解压的扩展程序"，选择解压的文件夹
5. 固定扩展图标到工具栏（可选）

### 2️⃣ 立即开始使用
安装后可以立即使用以下功能：
- ✅ 快速链接（DataSuite 快速入口）
- ✅ 代办事项
- ✅ 实用工具
- ✅ Code Helper
- ✅ 代码片段

### 3️⃣ 配置 Google Calendar（可选）
如需使用"今日日程"功能：
1. 点击扩展中的"今日日程"标签
2. 点击"⚙️ 配置"按钮
3. 按照页面提示完成 OAuth 配置
4. 详细步骤见 `OAUTH_SETUP_QUICK.md`

**提示：这是一次性配置，配置后所有功能都可正常使用！**

### 4️⃣ 查看完整使用指南
双击打开 `USER_GUIDE.html`，查看详细的功能说明、常见问题等。
