# 安装指南

## 快速开始

### 1. 准备图标文件（可选）

进入 `images` 文件夹，按照说明创建PNG格式的图标文件。如果没有图标，扩展仍可正常运行。

### 2. 在Chrome中加载扩展

1. 打开Chrome浏览器
2. 地址栏输入 `chrome://extensions/` 并回车
3. 打开右上角的 **"开发者模式"** 开关
4. 点击 **"加载已解压的扩展程序"** 按钮
5. 选择 `SPX_Helper` 文件夹
6. 点击 **"选择"**

### 3. 固定扩展到工具栏

1. 点击浏览器工具栏的拼图图标 🧩
2. 找到 **"值班助手"**
3. 点击图钉 📌 图标将其固定到工具栏

### 4. 开始使用

点击工具栏中的值班助手图标，开始使用！

## 功能快速上手

### 记录笔记
1. 在"快速笔记"区域输入内容
2. 点击"保存笔记"
3. 笔记会自动保存并显示时间

### 设置提醒
1. 输入提醒内容
2. 设置多少分钟后提醒
3. 时间到了会收到系统通知

### 管理链接
1. 添加常用的工作链接
2. 点击链接快速打开
3. 方便访问常用网站

### 值班计时
1. 点击"开始值班"开始计时
2. 实时查看值班时长
3. 结束后点击"结束值班"

## 疑难解答

### 问题：扩展无法加载
- 确认已开启"开发者模式"
- 检查文件夹路径是否正确
- 查看是否有文件缺失

### 问题：通知不弹出
- 检查Chrome通知权限
- 系统通知设置是否开启
- 勿扰模式是否关闭

### 问题：数据丢失
- 不要清除浏览器数据
- 定期导出重要笔记
- 考虑实现云端备份

## 更新扩展

修改代码后：
1. 回到 `chrome://extensions/`
2. 找到"值班助手"
3. 点击刷新按钮 🔄
4. 重新打开扩展即可看到更新
