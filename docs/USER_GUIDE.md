## 📥 安装步骤

### 1. 获取扩展包

下载 `SPX_Helper_v2.0.0.zip` 文件

### 2. 解压文件

将zip文件解压到任意位置，例如：
```
~/Downloads/SPX_Helper_v2.0.0/
```

⚠️ **重要**: 不要删除解压后的文件夹，扩展会一直从这个位置运行

### 3. 在Chrome中安装

#### 步骤 3.1: 打开扩展管理页面
在Chrome地址栏输入：
```
chrome://extensions/
```
或点击菜单：`⋮` → `扩展程序` → `管理扩展程序`

#### 步骤 3.2: 开启开发者模式
在页面右上角，开启 **开发者模式** 开关

#### 步骤 3.3: 加载扩展
1. 点击左上角 **"加载已解压的扩展程序"** 按钮
2. 选择解压后的 `build` 文件夹
3. 点击"选择"

#### 步骤 3.4: 确认安装
扩展应该出现在列表中，显示：
- ⚡ SPX Helper
- Shopee大数据开发助手
- v2.0.0

### 4. 固定到工具栏（推荐）

1. 点击Chrome工具栏右侧的 🧩 拼图图标
2. 找到 "⚡ SPX Helper"
3. 点击📌图标固定

## 🎯 快速开始

### 首次使用

点击工具栏的 ⚡ 图标，你会看到5个标签：

1. **🔗 快速链接** - 访问常用平台
2. **📋 代码片段** - SQL模板库
3. **🛠️ 实用工具** - 时区转换、JSON、命名转换
4. **💬 Code Helper** - SQL助手（需配置）
5. **✅ 待办** - 任务管理

### 核心功能

#### 1. 快速链接
- 预置了DataSuite、常用文档、开发工具等链接
- 可以添加/编辑/删除分类和链接
- 右键点击分类名可以编辑或删除

#### 2. 代码片段
- 预置了ClickHouse、Presto、Spark常用SQL
- 一键复制代码
- 可以添加自己的代码片段
- 右键点击分类可以管理

#### 3. 实用工具

**⏰ 时区转换**
- 时间戳转多地区时间（SG/VN/ID/MY/PH/TH/CN）
- 时间转时间戳

**📋 JSON工具**
- 格式化JSON
- 压缩JSON
- 验证语法
- 统计信息（字符数、层级、对象数、数组数）

**🔤 命名转换**
- 驼峰命名 (camelCase)
- 帕斯卡命名 (PascalCase)
- 下划线命名 (snake_case)
- 短横线命名 (kebab-case)
- 常量命名 (CONSTANT_CASE)
- 支持批量转换

#### 4. Code Helper
- SQL语法修正
- SQL优化建议
- 支持Presto、Spark SQL、ClickHouse
- **注意**: 已内置配置，直接使用即可

#### 5. 待办列表
- 创建/编辑/删除任务
- 设置优先级（高/中/低）
- 分类管理（开发、调试、优化等）
- 截止日期提醒
- 搜索和过滤

### 方法1: 手动更新
1. 下载新版本的zip文件
2. 解压到新位置
3. 在 `chrome://extensions/` 删除旧版本
4. 加载新版本

### 方法2: 刷新更新
如果只是修复了bug，开发者可能提供更新文件：
1. 替换旧文件
2. 在 `chrome://extensions/` 点击 🔄 刷新按钮

⚠️ **数据备份**: 更新前，重要数据请备份

## ⚙️ 配置说明

### 权限说明

扩展需要以下权限：

| 权限 | 用途 |
|------|------|
| `storage` | 保存你的链接、代码片段、TODO等数据 |
| `alarms` | 定时提醒（未来功能） |
| `notifications` | 桌面通知 |
| `activeTab` | 获取当前标签页信息 |
| `https://smart.shopee.io/*` | 访问Smart平台API（Code Helper） |

### 数据存储

所有数据保存在浏览器本地存储中：
- 快速链接
- 代码片段
- TODO列表
- 聊天历史

**不会上传到任何服务器**（除了Code Helper功能调用Smart API）

## 🆘 常见问题

### Q: 安装后图标不显示？
**A**: 点击工具栏的🧩拼图图标，将SPX Helper固定到工具栏

### Q: 提示"此扩展程序可能已损坏"？
**A**: 
1. 确保解压了完整的文件
2. 确保选择的是 `build` 文件夹
3. 尝试重新解压和安装

### Q: Code Helper无法使用？
**A**: 
1. 检查网络连接
2. 确保可以访问 `smart.shopee.io`
3. 配置已内置，无需手动设置

### Q: 数据会丢失吗？
**A**: 
- 正常情况下数据持久保存
- 卸载扩展会清除所有数据
- 建议定期备份重要的代码片段

### Q: 更新后数据丢失了？
**A**: 
- 更新前数据会保留
- 如果清除了浏览器缓存可能丢失
- 建议更新前导出重要数据

### Q: 支持其他浏览器吗？
**A**: 
支持所有基于Chromium的浏览器：
- ✅ Google Chrome
- ✅ Microsoft Edge
- ✅ Opera
- ✅ Brave
- ❌ Firefox（暂不支持）
- ❌ Safari（暂不支持）

### Q: 如何卸载？
**A**: 
1. 访问 `chrome://extensions/`
2. 找到 SPX Helper
3. 点击"删除"按钮
4. 确认删除

### Q: 扩展会收集我的数据吗？
**A**: 
- 不会！所有数据都保存在本地
- Code Helper功能会发送SQL到Smart平台
- 其他功能完全离线工作

## 💡 使用技巧

### 技巧1: 批量添加链接
在"快速链接"中，可以先创建分类，再批量添加链接

### 技巧2: 代码片段模板化

### 技巧3: 命名转换批量处理
在"命名转换"工具中，每行输入一个变量名，一次转换多个

### 技巧4: JSON工具快速验证
复制API响应到JSON工具，快速验证格式和查看统计

### 技巧5: TODO分类使用
- 开发任务 - 新功能开发
- 问题排查 - Bug修复
- 性能优化 - SQL优化等
