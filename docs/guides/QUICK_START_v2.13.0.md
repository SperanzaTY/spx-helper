# SPX Helper v2.13.0 - 快速启动指南

## 🚀 立即开始

### 1. 安装/更新扩展
```
chrome://extensions/ → 刷新 SPX Helper
```

### 2. 打开设置
```
扩展弹窗 → 🔧 实用工具 → App 工具 → 🔍 API 数据溯源 → ⚙️ 功能设置
```

### 3. 配置（可选）
- **文本选取开关**: 默认开启 ✅（日常不需要可关闭）
- **API 过滤条件**: 默认 `api_mart`（可自定义）

### 4. 开始使用
1. 打开任意 FMS 页面
2. 选择页面上的文字或数字
3. 点击浮动按钮 "🔍 查找来源"
4. 查看结果

---

## 💡 推荐配置

### 场景 1: FMS 后台开发（默认配置）
```
✅ 文本选取: 开启
📝 过滤条件: api_mart
```
**适用：** 日常开发，性能最佳

### 场景 2: 不需要查找数据源
```
⏸️ 文本选取: 关闭
📝 过滤条件: api_mart
```
**适用：** 避免浮动按钮干扰工作

### 场景 3: 全栈调试
```
✅ 文本选取: 开启
📝 过滤条件: api_mart, /api/bff/, graphql
```
**适用：** 需要追踪多个 API 来源

### 场景 4: 临时全量分析
```
✅ 文本选取: 开启
📝 过滤条件: (留空)
```
**适用：** 全量拦截所有接口（性能影响大，仅临时使用）

---

## 🎯 核心功能

### ✅ 已有功能
1. **文本选取查找** - 拖动选择文字，快速查找 API 来源
2. **API 自动拦截** - 自动记录所有 API 请求
3. **格式化匹配** - 支持货币、千分位等格式
4. **字段映射分析** - UI 字段与 API 字段对应关系
5. **导出功能** - CSV/JSON 格式导出

### 🆕 新增功能（v2.13.0）
6. **文本选取开关** - 可随时开启/关闭文本选取功能
7. **API 过滤自定义** - 自定义拦截哪些接口

## 🐛 遇到问题？

### 常见问题速查
1. **修改过滤条件后没生效** → 刷新页面
2. **开关切换后没反应** → 刷新页面让 content.js 重新加载
3. **控制台没有日志** → 重新加载扩展
4. **浮动按钮不显示** → 检查文本选取开关是否开启

### 查看详细文档
```
API_TRACKER_SETTINGS_GUIDE.md → 常见问题章节
```
