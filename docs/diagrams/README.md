# 图表（draw.io）

| 文件 | 说明 |
|------|------|
| `hub-driver-overview-v2-lineage.drawio` | **主文件**：Driver Overview v2 完整读/写血缘（API Mart、biz_sql、CK 事实/维表、写链路、Presto/Hive/GSheet 排查侧） |
| `hub-driver-overview-v2-lineage.drawio.xml` | 与 `.drawio` 内 `mxGraphModel` **同源**，便于 diff / 合并 |

用 [draw.io / diagrams.net](https://app.diagrams.net) 或 VS Code 的 Draw.io 插件 **直接打开仓库里的 `hub-driver-overview-v2-lineage.drawio`** 即可看到完整图。

**说明：** Cursor 里的 **drawio MCP 浏览器** 往往只显示会话里创建的**占位/简化图**，与仓库文件**不一定一致**。要看详细链路请以本目录下的 `.drawio` 为准，勿依赖 MCP 预览。
