# 📂 分类管理功能说明

## ✨ 新功能概览

现在**快速链接**和**代码片段**的分类都可以完全自定义了！

---

## 🔗 快速链接 - 分类管理

### 功能特性

✅ **查看分类**
- 所有链接按分类分组显示
- 每个分类都有独立的标题和链接列表
- 分类可以包含emoji图标

✅ **添加分类**
- 点击右上角的 `➕ 添加分类` 按钮
- 输入分类名称（可以包含emoji）
- 新分类会自动添加到底部

✅ **编辑分类**
- 悬停在分类标题上
- 点击 `✏️` 编辑按钮
- 修改分类名称

✅ **删除分类**
- 悬停在分类标题上
- 点击 `🗑️` 删除按钮
- ⚠️ **注意**: 删除分类会同时删除该分类下的所有链接

### 使用示例

```
原有分类：
⚡ 实时计算  → 可编辑
💾 离线计算  → 可编辑  
🏢 Shopee平台 → 可编辑

添加新分类：
🔧 开发工具
📊 监控平台
🎯 业务系统
```

---

## 📋 代码片段 - 分类管理

### 功能特性

✅ **动态过滤器**
- 过滤按钮根据分类动态生成
- "全部" 按钮查看所有片段
- 点击分类名称只显示该分类的片段

✅ **添加分类**
- 在过滤器栏右侧点击 `➕ 添加分类` 按钮
- 输入分类名称
- 自动生成分类ID（小写+下划线）
- 新分类会立即出现在过滤器和选择器中

✅ **编辑分类**
- 悬停在任意分类按钮上
- 点击 `✏️` 编辑按钮
- 修改分类名称

✅ **删除分类**
- 悬停在任意分类按钮上
- 点击 `🗑️` 删除按钮
- 系统会显示该分类下有多少个代码片段
- ⚠️ **注意**: 删除分类会同时删除该分类下的所有代码片段

✅ **分类验证**
- 自动检查分类是否已存在
- 防止重复添加
- 删除当前查看的分类会自动切换到"全部"视图

### 使用示例

```
原有分类：
ClickHouse → 可用
Flink SQL  → 可用
Spark      → 可用
Hive       → 可用
Presto     → 可用

添加新分类：
Trino
ClickHouse UDF
Flink DataStream
自定义工具
```

## 💾 数据存储结构

### 链接分类
```javascript
linkCategories: [
  { id: 'realtime', name: '⚡ 实时计算', order: 1 },
  { id: 'offline', name: '💾 离线计算', order: 2 },
  { id: 'shopee', name: '🏢 Shopee平台', order: 3 }
]
```

### 代码片段分类
```javascript
snippetCategories: [
  { id: 'clickhouse', name: 'ClickHouse', order: 1 },
  { id: 'flink', name: 'Flink SQL', order: 2 },
  { id: 'spark', name: 'Spark', order: 3 },
  { id: 'hive', name: 'Hive', order: 4 },
  { id: 'presto', name: 'Presto', order: 5 }
]
```

### 字段说明
- `id`: 唯一标识符，用于关联链接/片段
- `name`: 显示名称，支持emoji和中文
- `order`: 显示顺序，数字越小越靠前

## 🎯 快速开始

### 1. 添加链接分类

1. 打开 "快速链接" 标签
2. 点击右上角 `➕ 添加分类`
3. 输入分类名称（如：`🔧 开发工具`）
4. 新分类立即显示
5. 在下方表单中选择新分类并添加链接

### 2. 添加代码片段分类

1. 打开 "代码片段" 标签
2. 在过滤器栏右侧点击 `➕ 添加分类`
3. 输入分类名称（如：`Trino`）
4. 新分类出现在过滤器和下拉菜单
5. 添加代码片段时选择新分类

### 3. 编辑/删除分类

**链接分类**：
- 悬停在分类标题上 → 出现 ✏️ 和 🗑️ 按钮
- 点击对应按钮进行操作

**代码片段分类**：
- 悬停在任意分类按钮上 → 出现 ✏️ 和 🗑️ 按钮
- 点击 ✏️ 编辑分类名称
- 点击 🗑️ 删除分类（会显示该分类下有多少个片段）

---

## ⚠️ 注意事项

1. **删除链接分类会删除该分类下的所有链接**
   - 请谨慎操作
   - 建议先把重要链接移动到其他分类

2. **删除代码片段分类会删除该分类下的所有片段**
   - 删除前系统会显示该分类下的片段数量
   - 需要二次确认才能删除
   - 删除后会自动切换到"全部"视图

3. **分类ID唯一性**
   - 代码片段分类ID由名称自动生成
   - 相同名称会提示"该分类已存在"

4. **首次使用**
   - 扩展会自动初始化默认分类
   - 可以直接修改或删除默认分类
