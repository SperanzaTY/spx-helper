# SPX Helper - Shopee 大数据开发助手

## 简介

SPX Helper 是一款专为 Shopee 大数据开发工程师设计的 Chrome 浏览器扩展工具，旨在提升日常工作效率。集成了快速链接、日程管理、记事本、时区转换、SQL助手等多项实用功能。

---

## 主要功能

### 1. 今日日程

- 集成 Google Calendar，实时显示今日日程安排
- 快速查看待办事项看板（待办/进行中/已完成）
- 支持折叠已结束的日程

### 2. 快速链接

- 内置 DataSuite 快速入口（Datamap、Datahub、Studio、Scheduler、Flink）
- 预设常用开发工具链接（JIRA、DMS、GitLab、Apollo、Grafana 等）
- 预设常用文档链接（值班安排、Data Map、Mart Design 等）
- 支持自定义添加/编辑/删除链接
- 支持分类管理和排序

### 3. 记事本

- 支持创建、编辑、删除笔记
- 支持 Markdown 语法渲染
- 支持按分类筛选（工作笔记/学习笔记/日常记录）
- 支持关联待办事项
- 支持单独导出笔记为 Markdown 或纯文本格式

### 4. 实用工具

**时区转换**
- 时间戳转换为各市场本地时间（SG/VN/ID/MY/PH/TH/CN/BR）
- 本地时间转换为时间戳
- 一键复制转换结果

**JSON 工具**
- JSON 格式化/压缩
- JSON 语法校验
- 显示字符数和行数统计

**命名转换**
- 驼峰命名 (camelCase)
- 帕斯卡命名 (PascalCase)
- 下划线命名 (snake_case)
- 短横线命名 (kebab-case)
- 常量命名 (CONSTANT_CASE)

### 5. Code Helper (SQL 助手)

- 基于 Smart API 的 SQL 智能助手
- 支持 SQL 语法纠错和优化建议
- 支持 Presto、Spark SQL、ClickHouse 等多种方言
- 对话历史自动保存

### 6. 待办管理

- 看板式任务管理（待办/进行中/已完成）
- 支持设置优先级（高/中/低）
- 支持设置截止日期
- 支持任务分类
- 支持搜索和筛选

## 安装指南

### 第一步：申请测试权限

由于日历功能使用了 Google Calendar API，需要先添加测试用户权限。

请联系 tianyi.liang，提供你的 Shopee 邮箱地址，添加为测试用户后方可正常使用日历功能。

### 第二步：安装扩展

**方法一：拖拽安装（推荐）**

1. 下载 `SPX_Helper_v2.3.0.zip` 文件
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启右上角的「开发者模式」
4. 将 zip 文件直接拖拽到扩展管理页面
5. 点击「添加扩展程序」确认安装

**方法二：解压安装**

1. 下载并解压 `SPX_Helper_v2.3.0.zip` 到本地目录
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择解压后的 `SPX_Helper` 文件夹

### 第三步：固定到工具栏

安装完成后，点击浏览器工具栏的拼图图标，找到 SPX Helper，点击图钉图标将其固定到工具栏，方便快速访问。

### 第四步：授权日历（可选）

首次使用日历功能时，点击「授权访问 Google Calendar」按钮，使用 Shopee 账号登录并授权即可。

## 使用说明

### 快速链接

- 点击链接卡片即可在新标签页打开
- 点击「添加链接」可自定义新链接
- 右键点击分类标签可编辑或删除分类
- 使用上下箭头可调整链接顺序

### 记事本

- 点击「新建」创建新笔记
- 点击笔记卡片查看详情
- 在详情页点击「编辑」可修改内容
- 在详情页点击「复制」可导出当前笔记（支持 Markdown 或纯文本）
- 支持 Markdown 语法：标题(#)、粗体(**)、斜体(*)、列表(-)、代码(`)、引用(>)
- 可选择关联到某个待办事项

### 时区转换

- 输入时间戳（秒），自动显示各市场对应时间
- 点击「当前时间」获取当前时间戳
- 点击「复制」按钮复制对应时区的时间

### Code Helper

- 在输入框中输入 SQL 或问题
- 按 Enter 发送，Shift+Enter 换行
- 支持粘贴报错信息进行分析
- 对话内容会自动保存

### 待办管理

- 点击「添加任务」创建新待办
- 拖动任务卡片可改变状态
- 点击任务可查看详情和编辑
- 使用搜索框可快速筛选任务

## 常见问题

**Q: 扩展图标在哪里？**

A: 点击浏览器工具栏右侧的拼图图标，找到 SPX Helper，点击图钉固定到工具栏。

**Q: 数据会丢失吗？**

A: 所有数据保存在浏览器本地存储中，卸载扩展或清除浏览器数据会导致数据丢失。建议定期使用导出功能备份重要笔记。

**Q: 如何更新扩展？**

A: 下载新版本 zip 文件，在扩展管理页面先移除旧版本，再安装新版本。

**Q: 日历显示不了怎么办？**

A: 请先联系 tianyi.liang 添加测试用户权限，然后在扩展中点击「授权访问 Google Calendar」完成授权。
