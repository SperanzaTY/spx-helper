# SPX Helper v2.6.6

## 主要更新

- ✅ 修复更新检查逻辑，支持从 GitHub Releases API 检查更新
- ✅ 新增 GitHub 仓库配置功能
- ✅ 优化更新检查提示信息
- ✅ 修复本地文件检查无法获取远程版本的问题

## 安装方法

1. 下载 `SPX_Helper_v2.6.6.zip` 文件
2. 解压到本地目录
3. 打开 Chrome，访问 `chrome://extensions/`
4. 开启"开发者模式"
5. 点击"加载已解压的扩展程序"
6. 选择解压后的文件夹

## 更新说明

如果你是从旧版本更新：
1. 建议先导出数据（设置 → 导出数据）
2. 移除旧版本扩展
3. 安装新版本
4. 导入之前导出的数据
